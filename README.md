### 📲 Быстрые инструкции

**Android**  
v2rayNG / Hiddify / NekoBox / Matsuri → «+» → Import from URL → вставь ссылку

**iOS**  
V2Box / Shadowrocket / Streisand → Add Subscription → вставь

**Windows / macOS / Linux**  
Hiddify / Throne / v2rayN / Clash Verge → Import / Subscribe → URL

Выбирай сервер с пингом < 150–200 мс и низкими потерями.
